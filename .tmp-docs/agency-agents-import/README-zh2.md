## 🚀 快速开始

### 方式一：Claude Code（推荐）

```bash
cp -r agency-agents/* ~/.claude/agents/
```

然后在 Claude 对话中：
> "使用前端开发工程师代理帮我构建 React 组件"

### 方式二：其他工具

```bash
./scripts/convert.sh    # 生成集成文件
./scripts/install.sh    # 交互式安装
```

支持：Cursor · Aider · Windsurf · Gemini CLI · OpenCode · Antigravity

### 方式三：直接参考

浏览下方代理列表，复制/适配你需要的提示词。
