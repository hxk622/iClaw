# 05 decisions

## 目标

这里不找“精彩观点”，只找：

- 真实决策
- 配置调整
- 风险处理
- 误判与修正

因为商店里的 expert 最终得回答：

> “在真实世界里，这套框架会怎么落地？”

## 候选决策线索

### Ray Dalio / Bridgewater

#### 线索 1：把宏观问题拆成机制问题

- 不是直接给市场结论
- 而是先问：
  - 债务周期怎么走
  - 信贷怎么扩张 / 收缩
  - 货币和利率如何影响资产

**可提炼价值**：
- 适合作为 skill 的“先变量后结论”工作流

#### 线索 2：强调分散与情景覆盖

- Dalio 最有代表性的不是押对一次宏观方向
- 而是把配置做成能覆盖多种环境的系统

**可提炼价值**：
- skill 不能只说“看多/看空”
- 必须讲：
  - 情景
  - 覆盖
  - 脆弱点

### Howard Marks

#### 线索 3：把风险控制放在收益前面

- 从 Oaktree memo 的一贯风格看，Marks 反复强调：
  - 避免糟糕年份
  - 不要过度确信
  - 周期位置决定可承受风险

**可提炼价值**：
- skill 的决策建议必须把风险前置

#### 线索 4：反对“宏观预测式自信”

- Marks 一类投资人最值得借的不是押方向，而是：
  - 对不确定性的表达纪律
  - 对概率和赔率的尊重

**可提炼价值**：
- 这个 expert 的核心不是预测未来
- 而是解释在不同未来下，配置如何更稳健

## 可直接写进 skill 的决策启发式 seed

1. 宏观配置先做情景拆分，再做资产映射
2. 如果结论依赖单一变量，这不是稳健配置
3. 如果一个判断无法说明失效条件，它还不是配置结论
4. 大类资产不是同时追求“都涨”，而是承担不同角色
5. 风险控制的目标不是避免波动，而是避免不可恢复损失

## 当前判断

这个 skill 在 decisions 维度最有价值的，不是某一次具体押注，而是：

> **把配置变成一套面对不确定性的纪律系统**

这是和普通“宏观观点专家”最该拉开的差异。
