- 本次发布目标：推进 `1.0.2.202604090003` 正式版本，发布共享服务与两个品牌桌面端，并切为强更。

## 7. 风险点

- 当前正式产物目录以 Windows `x64` 为主，本次发布不覆盖 macOS 产物
- 强更采用 `mandatory + required_now`，旧版本用户会被立即阻断继续使用
- 若下载站点未切到最新 manifest，会出现“下载页版本号已更新但文件仍指向旧包”的偏差
- Windows runtime 远端文件如再次变更但未同步 SHA，本次桌面包仍会被后续更新链路阻断
