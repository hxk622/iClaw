# iClaw 知识库 V1

本目录沉淀 iClaw「知识库」模块的 V1 产品定义。知识库是 iClaw 客户端内的统一知识工作台，采用固定三栏结构：

- 第一栏：对象导航
- 第二栏：对象视图
- 第三栏：Chat 协作

第一栏固定使用三层知识生产结构：

- Raw / 素材
- Graph / 图谱
- Output / 成果

## 关键结论

### 1. 知识库是 iClaw 主框架，不是 OpenAlpha 思维库的镜像

OpenAlpha extension 提供的是浏览器侧采集能力，不是产品母体。iClaw 内部统一以「知识库」为心智锚点，不再并行维护一套“思维库”和一套“知识库”。

### 2. OpenAlpha extension 在 iClaw 体系中只承担 Raw ingest 职责

插件负责：

- 页面捕获
- 滑选摘录
- 视频时间戳摘录
- 本地暂存与去重
- 同步到 iClaw 素材层

插件不负责：

- 图谱编译
- 成果生成
- 客户端主知识库浏览
- 主聊天体系

### 3. V1 先做真素材，再做真图谱，再做真成果

V1 的最短闭环不是一次性做完完整知识 OS，而是按以下顺序推进：

1. Raw / 素材真实化
2. Graph / 图谱数据化 + 第二栏视图化
3. Output / 成果对象化
4. Output 反哺 Graph

### 4. 第三栏永远复用 iClaw 现有 Chat Surface

知识库第三栏不是独立聊天系统。选中的素材、图谱、成果只作为上下文输入给 iClaw 现有对话面板。

## V1 成功标准

如果只看一个最小成功标准，定义为：

> 用户通过本地上传或浏览器插件把真实 Raw 导入知识库；第一栏可以看到真实素材；第二栏可以看到真实详情；第三栏可以围绕这个对象进行上下文对话。

这代表知识库已经从概念页进入真实数据闭环。
