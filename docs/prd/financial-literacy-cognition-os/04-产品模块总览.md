# 04. 产品模块总览

## 模块地图

v1 的核心模块分为 7 个：

1. **Onboarding Diagnostic（入门诊断）**
2. **Cognition Curriculum（认知课程系统）**
3. **Model Diagrams（模型图解系统）**
4. **Simulation Lab（认知实验室）**
5. **IPS Builder（IPS 构建系统）**
6. **Review & Rewrite Loop（复盘重写系统）**
7. **Progress System（学习路径与进度机制）**

## 模块关系

```text
入门诊断
  -> 认知课程
  -> 模型图解 + 思维检查
  -> 情境模拟实验
  -> IPS 写作
  -> 失败后复盘与重写
  -> 学习进度推进
```

## 产品闭环

本产品不是“看完课就结束”，而是一个反复迭代的闭环：

1. 先识别用户当前认知缺口
2. 通过课程与图解建立底层模型
3. 用模拟场景暴露错误假设
4. 强迫用户把判断写成 IPS
5. 再通过失败与复盘重写 IPS
6. 最终形成可迁移到真实世界的认知操作系统
