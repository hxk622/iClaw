# 07. IPS 构建与复盘系统

## 为什么 IPS 是产品核心产物

IPS（个人资产投资备忘录）是认知外化的终点。

只有当用户把自己的判断系统写出来，我们才能知道：
- 他是否真正理解了资产角色
- 他是否有明确的风险边界
- 他是否提前写下了 If-Then 预案
- 他的逻辑是否自洽

## IPS 必须包含的内容

1. **资产角色定义**
   - 为什么持有某类资产
   - 它在整个系统中的角色是什么

2. **风险边界**
   - 可接受回撤
   - 时间周期
   - 不允许突破的原则

3. **If-Then 规则**
   - 如果出现 20% 回撤，我怎么处理
   - 如果某主题暴涨，我怎么判断是否参与
   - 如果市场长时间无反馈，我是否改变策略

4. **复盘规则**
   - 什么情况下要重写 IPS
   - 哪些信号意味着原假设失效

## 复盘与重写机制

模拟实验或测验暴露问题后，系统不能只提示“你错了”，而要推动：

1. 定位错误假设
2. 对照原 IPS 查看缺口
3. 要求用户重写相关条目
4. 记录修改前后差异

## 通过标准

最终不是看文笔，而是看逻辑是否：
- 不自相矛盾
- 有明确边界
- 能对应典型市场场景
- 能经得起复盘审计
