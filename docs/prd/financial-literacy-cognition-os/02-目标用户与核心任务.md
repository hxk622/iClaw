# 02. 目标用户与核心任务

## 目标用户

v1 默认面向“张晓波类”用户：

- 年龄：25–40 岁
- 特征：中产、职业稳定、逻辑思维较强
- 行为：订阅财经内容、刷公众号/播客/短视频、追热点板块
- 困境：知道很多金融术语，但没有形成自己的认知框架

## 典型症状

- 遇到“利好”就冲，遇到“暴跌”就慌
- 频繁寻找“今天买什么”的答案
- 学了很多内容，却越来越依赖他人的判断
- 有努力学习的幻觉，但没有独立决策能力

## 核心用户任务（Core Job）

> 帮我停止把投资当成预测游戏，转而建立一套书面化、可验证、可复盘的概率决策系统。

## 90 天目标转变

用户要从：
- 行情狩猎者
- 资讯追逐者
- 情绪驱动的临场决策者

转变为：
- 系统管理官
- 会先提问题、再做判断的人
- 基于赔率、风险边界和预案行事的人

## 关键行为变化

- 停止每天高频看盘和追热点
- 不再问“现在追还来得及吗”
- 能区分“自己懂的机会”和“不是自己该赚的钱”
- 能在波动来临前写好应对规则，而不是波动之后靠意志力硬扛
