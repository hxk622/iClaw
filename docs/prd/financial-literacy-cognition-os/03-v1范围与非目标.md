# 03. v1 范围与非目标

## v1 范围

v1 聚焦于“认知系统安装”的最小闭环：

1. 用户认知诊断
2. 结构化认知课程
3. 模型图解与思维检查
4. 模拟实验室
5. IPS（个人资产投资备忘录）写作
6. 复盘与重写
7. 基础进度机制

## v1 不做什么

### 明确非目标
- 不做荐股
- 不替用户下单
- 不承诺收益
- 不做“今天买什么”的行动建议
- 不做重交易、重投顾、重账户聚合工具

### 暂缓项
- 自动生成视频讲解作为首要形态
- 社群/社区功能
- 导入真实持仓做深度资产助手
- 实时市场跟随型个性化推荐
- 高阶机构投资研究内容

## v1 边界原则

- 所有个性化都应服务于认知升级，而不是服务于“更容易上瘾”
- 所有交互都应服务于理解风险、概率、预案，而不是刺激频繁交易
- 游戏化只能作为认知实验机制的一部分，不能演变为单纯打卡产品
