# 14. 课程体系 Syllabus 详细大纲

## 目标

这套 syllabus 的目标不是覆盖所有金融知识，而是完成 v1 的核心任务：

> 帮用户建立一套可书面化、可验证、可复盘的金融认知操作系统。

## v1 课程结构

建议拆成 8 个模块，覆盖 90 天学习闭环。

---

## Module 1：金融的本质

### 核心问题
- 金融到底是什么？
- 为什么金融不是“预测未来”，而是“风险定价 + 跨时空资源配置”？

### 学习目标
- 建立金融的第一性定义
- 理解资产、风险、时间、预期之间的基本关系
- 打破“投资=猜涨跌”的直觉

### 关键概念
- 风险定价
- 时间价值
- 资源配置
- 概率与不确定性

### 输出要求
- 用自己的话解释“金融的本质是什么”
- 写下自己过去对金融的一个误解

---

## Module 2：收益、风险与概率

### 核心问题
- 为什么高收益一定伴随高不确定性？
- 为什么“高收益、低风险、稳赚钱”通常是错觉？

### 学习目标
- 区分收益率、波动、风险、赔率、胜率
- 理解风险不是单纯“会不会跌”，而是“结果分布的不确定性”

### 关键概念
- 收益与风险
- 胜率 vs 赔率
- 结果分布
- 尾部风险

### 输出要求
- 解释一个“看起来稳赚但其实高风险”的例子
- 在 IPS 中补充自己的风险语言

---

## Module 3：价格、价值与估值

### 核心问题
- 为什么资产价格经常偏离其长期价值？
- 市场价格由什么驱动？

### 学习目标
- 理解价格与价值不是同一件事
- 理解估值与预期、流动性、风险偏好之间的关系

### 关键概念
- 估值
- 预期差
- 安全边际
- 情绪定价

### 输出要求
- 分析一个“价格涨得快但不代表价值提升”的例子
- 写下自己如何判断“贵”和“便宜”

---

## Module 4：周期、流动性与宏观传导

### 核心问题
- 为什么宏观变量会影响资产价格？
- 利率、流动性、通胀预期是如何传导的？

### 学习目标
- 建立从宏观变量到资产反应的因果链
- 不再把“宏观新闻”当成孤立热点

### 关键概念
- 利率
- 流动性
- 通胀预期
- 风险偏好
- 信用扩张/收缩

### 输出要求
- 画出一个“流动性变化 -> 风险资产估值变化”的简图
- 解释一个宏观变量如何影响自己的资产判断

---

## Module 5：资产配置与相关性

### 核心问题
- 为什么分散不等于平均配置？
- 为什么资产角色要先于收益预期？

### 学习目标
- 理解不同资产在组合中的作用
- 理解相关性、对冲、暴露、集中风险

### 关键概念
- 资产角色
- 相关性
- 对冲
- 风险暴露
- 组合思维

### 输出要求
- 为 3 类常见资产写出“角色定义”
- 在 IPS 中填写资产角色区块

---

## Module 6：行为偏差与认知陷阱

### 核心问题
- 为什么明明懂道理，还是会在关键时刻犯错？
- 哪些偏差最容易破坏投资系统？

### 学习目标
- 识别 FOMO、损失厌恶、锚定、确认偏误、过度自信
- 知道偏差不是道德问题，而是系统问题

### 关键概念
- FOMO
- 损失厌恶
- 锚定效应
- 确认偏误
- 过度自信

### 输出要求
- 标记自己最常见的两个偏差
- 对应写出一条防错规则

---

## Module 7：预案、纪律与 If-Then 系统

### 核心问题
- 为什么系统先于判断？
- 为什么没有预案的人，最终一定靠情绪做决策？

### 学习目标
- 理解“预案”是认知系统的执行层
- 学会把抽象认知写成 If-Then 规则

### 关键概念
- 预案
- 决策规则
- 纪律
- 情境响应

### 输出要求
- 至少写出 3 条 If-Then 规则
- 把一条模糊直觉翻译成可执行语句

---

## Module 8：复盘、修正与系统升级

### 核心问题
- 为什么错误本身不是问题，拒绝升级才是问题？
- 如何根据证据修正自己的系统？

### 学习目标
- 学会用复盘而不是情绪解释错误
- 学会根据实验和结果修改 IPS

### 关键概念
- 复盘
- 假设修正
- 版本升级
- 审计思维

### 输出要求
- 对一次错误决策做结构化复盘
- 更新 IPS 一个关键条目并说明原因

---

## 每个模块的统一结构

每个模块都应包含：
1. 第一性原理讲解
2. 模型图解
3. 小测/推理题
4. 场景应用
5. 对 IPS 的影响提示
6. 下一步建议

## 90 天节奏建议

### 第 1–30 天
- Module 1–4
- 以认知建立为主

### 第 31–60 天
- Module 5–6
- 引入实验室和偏差识别

### 第 61–90 天
- Module 7–8
- 完成 IPS 定稿与复盘闭环

## v1 教学完成标准

用户完成课程后，应至少能做到：
- 用自己的话解释金融不是预测游戏
- 区分风险、收益、赔率和胜率
- 描述至少 3 类资产的角色
- 识别自己常见的行为偏差
- 写出 3 条以上 If-Then 规则
- 形成一份初步可审计的 IPS
