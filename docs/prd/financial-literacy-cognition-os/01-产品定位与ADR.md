# 01. 产品定位与 ADR

## 产品定位

本产品定位为：

> **内容优先的金融认知操作系统（Financial Cognition OS）**  
> 通过结构化认知学习与模拟实验，帮助用户从“预测市场”转向“管理概率、风险与预案”。

## 为什么做

多数股民、基民的问题，不是信息太少，而是：

- 接收了太多碎片化信息
- 知道很多术语，但无法建立金融因果链条
- 把金融误解为“预测未来的游戏”
- 一遇到波动就回到情绪和 FOMO 驱动

因此，我们要解决的不是“教更多知识点”，而是**重建底层认知系统**。

## ADR

### Decision
构建一个 **Financial Cognition OS + Simulation Lab** 的 v1，核心不在荐股、交易或收益，而在认知升级与书面化决策系统形成。

### Drivers
- 根问题是金融底层理解浅，而非资讯缺乏
- 目标必须是行为层改变，而不是背会概念
- 验证标准必须能外化为可审计产物，而不是只看活跃/留存

### Alternatives Considered
1. **Content Academy**：课程 + 测验 + 进度
2. **Cognition OS + Simulation Lab**：课程 + 模拟 + IPS
3. **Hybrid Wealth Companion**：教育 + 投资助手/提醒/实时指导

### Why Chosen
选择第二种，因为它最贴合“认知系统安装”这一目标，同时能守住教育边界，不滑向投顾或交易产品。

### Consequences
- v1 会更硬核、更窄，但认知密度更高
- 模拟和 IPS 设计质量会成为成败关键
- 一部分只想要“财富密码”的用户会被主动过滤掉
