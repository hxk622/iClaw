# 金融专家分类与首批蒸馏样本

> 目标：为 `nuwa.skill` 蒸馏建立母结构，先定金融专家一级分类，再给出首批 4 个可执行蒸馏样本。  
> 原则：**已有专家全部保留**，不推翻现有商店，只做上层 taxonomy 升级与增量扩展。

## 1. 母结构原则

当前前端已经有一套方法论标签：

- 价值投资
- 红利收益
- 量化策略
- 宏观轮动
- 事件驱动
- 情绪背离
- 内部信号
- 成长挖掘
- 科技估值
- 组合诊断
- 其他

这些标签继续保留，但它们更适合作为 **二级方法标签**，而不是蒸馏母分类。

### 新的母结构

一级分类按 **市场 / 资产域** 建：

1. A股股票研究
2. 港股股票研究
3. 美股与海外股票研究
4. 基金 / ETF / 组合配置
5. 宏观 / 周期 / 资产配置
6. 债券 / 利率 / 固收
7. 黄金 / 大宗商品
8. 期货 / 衍生品 / 期权
9. 外汇 / 全球宏观交易
10. 加密资产 / 数字资产

二级标签继续沿用现有前端方法论标签体系。

## 3. 首批 4 个蒸馏样本

这 4 个样本用于跑通第一批 `nuwa.skill` 蒸馏流水线：

1. `global-macro-allocation-expert`
2. `gold-allocation-expert`
3. `bond-and-fixed-income-expert`
4. `a-share-growth-research-expert`

选择标准：

- 补现有专家商店空缺
- 用户能理解
- 边界清晰
- 适合被蒸馏成主题型 skill

## 4. 样本 A：global-macro-allocation-expert

### 一级分类

- 宏观 / 周期 / 资产配置

### 二级标签

- 宏观轮动
- 组合诊断

### 解决的问题

- 利率、通胀、流动性如何影响股票 / 债券 / 黄金 / 美元
- 大类资产该如何轮动
- 风险偏好变化下如何调整配置

### 蒸馏来源

#### 人物型主来源
- Ray Dalio（主）

#### 主题型补充
- 桥水式大类资产配置框架
- 流动性传导框架
- 宏观情景分析框架

### 边界

- 不做宏观事件点位预测
- 不做“押方向”式口号
- 强调情景、概率、资产角色和配置结构
