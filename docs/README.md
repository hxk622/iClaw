## 目录结构

- `architecture/`
  - 系统架构、运行时链路、控制面设计、客户端监控架构
- `design/`
  - 设计说明、交互规范、视觉与产品设计稿映射
- `plans/`
  - 实施计划、落地拆解、implementation plan
- `standards/`
  - 后台交互规范、流程规范
- `release/`
  - 发版标准、清单、矩阵、模板、桌面发版 SOP
- `reference/`
  - 通用参考资料、接口契约、决策记录
- `ops/`
  - 运维、基础设施、部署与采购相关文档
- `prd/`
  - 各业务线产品需求文档
- `presentations/`
  - 演示稿与 deck 产物
- `version_record/`
  - 版本记录与测试报告
- `archive/`
  - 临时、未分类或待整理材料
