# Figma -> iClaw v0 映射清单

更新时间：2026-03-03
设计来源：iClaw 当前前端稿与实现对照

## 标记说明

- `静态`：仅展示，不接后端逻辑
- `动态`：v0 要接真实接口
- `延后`：明确不在 v0 实现

## 映射表

| 区域/组件 | 状态 | v0 处理方式 |
|---|---|---|
| 左侧菜单（全部入口） | 静态 | 保留 UI，不接业务逻辑 |
| 左侧会话列表样式 | 静态 | 先用假数据/占位 |
| 聊天内容区（消息展示） | 动态 | 接 SSE 流并原样渲染 |
| 输入框 | 动态 | 绑定发送接口 |
| 发送按钮 | 动态 | 触发流式请求 |
| 消息加载态 | 动态 | 显示“生成中”状态 |
| 错误提示区域 | 动态 | 展示错误 + 重试 |
| 附件入口（若在设计内） | 动态 | 接 upload 接口 |
| 设置入口 | 延后 | 可隐藏或禁用 |
| 飞书/钉钉/IM 机器人配置 | 延后 | v0 不实现 |
| 主题切换入口 | 延后 | v0 默认 Light，不开放切换 |
| 高级模型/skill 操作入口 | 延后 | v0 不实现 |

## v0 页面行为补充

- 内容区渲染规则：后端返回什么展示什么，不做格式控制。
- 左侧交互规则：点击不触发真实导航，可做轻提示（可选）。
