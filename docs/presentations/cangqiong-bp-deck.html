01 / 愿景

我们的愿景：让人类专注创造，让 AI 以终端化、平台化、行业化的方式进入千行百业

我们要占的位置

  • 不是又一个聊天助手，而是 AI 终端入口
  • 不是单点模型能力，而是 可部署的智能体工作台
  • 不是纯软件订阅，而是 硬件 + 平台 + 行业服务 的新基础设施
  • 不是云端托管依赖，而是 用户可拥有的数据与算力载体

愿景落点

A

人人可拥有

AI 像电脑、手机一样成为个人与组织的长期资产。

B

人人可控制

权限、模型、数据、工作流由用户和企业掌握。

C

人人可部署

从家庭桌面到政企内网，统一能力、分级交付。

D

人人可受益

让 AI 真正创造效率，而不是制造新门槛。

02 / 问题

AI 能力越强，离普通用户和真实业务反而越远

01

部署鸿沟

技术供给快速迭代，但企业真正需要的是可开箱、可上线、可维护的现成方案。

02

安全鸿沟

云端集中化与高权限运行模式，难以满足数据主权、隐私保护与合规要求。

03

场景鸿沟

通用模型能力很强，但行业工作流、知识库、Skill 封装仍然稀缺。

04

成本鸿沟

长期 token 计费、高频调用与云端依赖，让中小企业和个人难以规模化使用。

今天真正缺的，不是再多一个模型，而是一个能把 算力、模型、工具、数据、工作流 变成可交付产品的系统。

05 / 技术亮点与领先性

技术领先性不在模型更大，而在于把模型变成可控、可部署、可规模化交付的系统

技术亮点

  • Local-first 架构：本地优先，兼顾离线运行、低延迟和数据主权
  • 混合模型调度：本地模型、私有模型、云模型按任务动态协同
  • Skill / Workflow 封装：让通用模型成为行业能力，而非一次性 prompt
  • 终端与平台协同：设备负责感知与执行，平台负责管理、分发与订阅
  • 工程化交付能力：权限、部署、回滚、私有化、安全策略形成企业可采购能力

相对云端 Agent 的领先性

更稳定

不依赖高权限云执行与持续联网。

更合规

适配政企、教育、法律、金融等敏感场景。

更低成本

高频任务不被 token 曲线持续侵蚀利润。

更易复制

行业模板、知识库、Skill 形成复用资产。

06 / 竞争壁垒

我们的护城河不在单点功能，而在“终端入口 + 本地安全 + 平台复用 + 行业落地”的组合

MOAT-1

终端入口壁垒

一旦 AI 成为设备而非网页插件，用户关系、数据关系、服务关系都会更深。

MOAT-2

安全与私有化壁垒

对企业和政府场景，隐私、安全、离线与可控不是附加项，而是采购前提。

MOAT-3

平台复用壁垒

Skill、工作流、知识库与模型编排越丰富，迁移成本越高，复制效率越强。

MOAT-4

行业落地壁垒

谁先沉淀场景模板、交付方法与渠道网络，谁就更难被纯模型公司替代。

这是一个 系统级优势，而不是一项功能上线就能被复制的优势。
