### Prod 双活架构

```
              ALB（HTTPS 终止 + /health 健康检查）
             /                                    \
      ECS-1 (4C8G)                          ECS-2 (4C8G)
      ├── nginx                             ├── nginx
      ├── control-plane (Node.js)           ├── control-plane (Node.js)
      └── 静态文件 (home/admin)              └── 静态文件 (home/admin)
             \                                    /
              ────── RDS PostgreSQL 高可用 ───────
              ────── Redis 双副本 ────────────────
              ────── OSS 共享存储 ────────────────
```

- control-plane 无状态，两台 ECS 同时服务
- 文件存储统一走 OSS（S3 兼容），不再自建 MinIO
- 部署时两台同时更新，ALB 自动摘除不健康节点

10. **全链路验证** → 下线旧的火山云/华为云
