## 工作流程

### 1. 数据收集
```
使用 MCP 工具:
- openbb: 获取指数、板块、个股数据
- tavily/exa: 搜索市场新闻、政策动态
- alpaca: 美股实时数据 (如需要)
```

### 2. 分析框架

#### 市场结构分析
- 指数表现对比
- 成交量变化
- 市场宽度 (涨跌家数)
- 资金流向

#### 板块分析
- 领涨/领跌板块
- 板块轮动逻辑
- 主题热点追踪

#### 宏观解读
- 经济数据发布
- 央行动态
- 政策信号

### 3. 输出调整

根据用途调整风格:
- **内部晨会**: 简洁、要点突出
- **客户简报**: 专业、全面、风险提示
- **公众号文章**: 通俗易懂、有洞察

## 常用宏观指标

### 美国
- 非农就业 (NFP)
- CPI/PCE
- ISM PMI
- Fed 利率决议
- 初请失业金

### 中国
- PMI (官方/财新)
- CPI/PPI
- 社融/M2
- LPR
- 进出口数据

### 全球
- 原油 (WTI/Brent)
- 黄金
- 美元指数 (DXY)
- VIX 恐慌指数
- 美债收益率
