# 外汇与经济日历工具包

用于获取 FX 与宏观事件相关数据，覆盖价格、汇率和经济发布日历。

## 适用场景

- 查询美元兑人民币、欧元兑美元、美元指数等外汇与汇率数据
- 获取主要宏观指标的发布时间、发布频率与历史序列
- 获取通胀、利率、就业、增长等宏观事件的跟踪数据

## 工作流程

1. 明确用户要的是哪一类
- 外汇行情 / 汇率
- 宏观指标时间序列
- 经济日历 / 发布时间

2. 选择查询维度
- 实时值
- 历史值
- 事件发布时间
- 同类指标对比

3. 调用数据源
- 优先使用 Alpha Vantage MCP 获取外汇与汇率数据
- 优先使用 FRED MCP 获取美国宏观指标与发布时间
- 对事件型补充搜索，可结合 Tavily / Serper

4. 输出要求
- 标明货币对、基准货币、时间区间与频率
- 对宏观指标标明国家、指标名称、统计口径与发布日期
- 如涉及未来事件，只输出已知日历安排，不做未经证实的推断

## 可交付内容

- 外汇实时与历史行情
- 主要宏观指标快照
- 经济日历清单
- 指标发布时间轴

## 注意事项

- 不同数据源的发布时间可能有时区差异，输出时统一标时区
- 经济日历属于动态信息，必须写明日期和来源
