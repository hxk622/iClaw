# 期货与贵金属数据工具包

用于获取期货、商品与贵金属相关数据，优先调用已接入的市场数据 MCP。

## 适用场景

- 查询黄金、白银、原油、铜等贵金属或大宗商品价格
- 查询股指期货、商品期货或连续合约的历史走势
- 对主力合约、近月合约、滚动合约做比较

## 工作流程

1. 明确资产类别
- 贵金属：黄金、白银
- 能源与商品：原油、天然气、铜等
- 期货：股指期货、商品期货、连续合约

2. 明确查询维度
- 实时行情
- 历史行情
- 区间涨跌
- 波动率与成交概况

3. 调用数据源
- 优先使用 Alpha Vantage MCP 获取贵金属与大宗商品数据
- 如需常见期货代码与行情补充，可使用 Yahoo Finance MCP

4. 输出要求
- 标明具体合约或资产代码
- 标明价格单位、币种、时间区间与频率
- 若为连续合约或近月合约，需明确口径

## 可交付内容

- 黄金 / 白银价格快照
- 商品与期货历史走势
- 主力合约对比表
- 波动与阶段表现摘要

## 注意事项

- 期货数据容易受到合约换月影响，连续合约与单一合约不能混用
- 贵金属和商品可能存在现货、期货、ETF 三种不同口径，输出时必须标明
