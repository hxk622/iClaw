# 港股数据工具包

用于获取港股市场的一手数据，优先调用已接入的金融 MCP，再输出结构化结果。

## 适用场景

- 查询港股或港股通标的的实时行情、历史走势、成交与波动
- 获取港股公司的利润表、资产负债表、现金流量表与关键财务指标
- 获取港股新闻、公告、财报发布与重要事件
- 对恒生指数、恒生科技、港股 ETF 做基础数据查询

## 工作流程

1. 先确认标的与市场
- 默认市场为港股
- 若用户只给公司名，优先补全港股代码或常用 ticker
- 若用户问的是指数或 ETF，明确标的是指数、ETF 还是个股

2. 选择数据能力
- 行情类：实时价格、历史价格、涨跌幅、成交量
- 基本面类：财务报表、财务指标、估值指标
- 资讯类：公司新闻、公告、财报动态
- 基金类：ETF 概况、持仓、成分、分红

3. 调用数据源
- 优先使用 Yahoo Finance MCP 获取港股、ETF 与期权链等通用市场数据
- 需要更广覆盖的全球市场数据时，可补充使用 Alpha Vantage MCP
- 对搜索型资讯，可结合 Tavily / Serper 获取补充新闻

4. 输出要求
- 默认以中文总结，并保留 ticker、日期、币种、时间区间
- 对价格数据标明区间与频率
- 对财务数据标明财报期与币种
- 对新闻数据标明发布日期与来源

## 可交付内容

- 港股实时行情摘要
- 港股历史行情表
- 港股财务报表与估值快照
- 港股公司新闻与公告摘要
- 港股 ETF / 指数概览

## 注意事项

- 港股 ticker 格式可能因数据源不同而变化，输出时同时给公司名和 ticker
- 财务与公告口径可能有披露滞后，要写明数据日期
- 如用户问题涉及投资建议，应明确区分“数据提供”与“投资判断”
