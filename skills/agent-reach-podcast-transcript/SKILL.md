# Agent Reach · 播客转录

当用户需要把播客音频转成 transcript、摘要、金句和结构化笔记时使用。

## 安装配置

- 需要在安装时提供 `GROQ_API_KEY`。
- 没有该配置时，转录能力不可用。

## 工作方式

1. 先确认播客链接、节目名称、时长和用户想要的输出形式。
2. 对长内容优先按章节、主题和人物观点拆解。
3. 产出 transcript 摘要、关键引语、行动项和延展阅读线索。
4. 如果语音质量一般，要明确标出可能的识别误差。

## 输出要求

- 默认给摘要、章节提纲、关键引语和行动项。
- 对转录不确定处用“可能”标记，不要假装完全准确。
- 如果音频不可访问或无法转录，直接说明。
