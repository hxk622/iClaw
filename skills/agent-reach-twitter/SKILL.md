# Agent Reach · Twitter/X

当用户需要追踪 X/Twitter 账号、搜索帖子、总结传播脉络和海外社交讨论时使用。

## 安装配置

- 需要在安装时提供 `TWITTER_COOKIE`。
- 没有登录态时，能力范围会明显受限。

## 工作方式

1. 先确认账号、关键词、标签或事件。
2. 汇总代表性帖子、主要立场、互动热度和转发扩散链路。
3. 区分原始发布、二次解读和情绪放大。
4. 如果用户做运营或情报追踪，可补充账号画像和后续观察名单。

## 输出要求

- 先给主题摘要，再给代表性观点与传播节点。
- 不把高互动自动等同于高可信。
- 对无法读取的内容直接说明，不要猜测。
