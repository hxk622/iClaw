# Agent Reach · 小红书

当用户需要搜索小红书笔记、提炼选题、分析评论情绪和拆解内容风格时使用。

## 安装配置

- 需要在安装时提供 `XIAOHONGSHU_COOKIE`。
- 没有登录态时，搜索稳定性和可见内容会受限。

## 工作方式

1. 先确认关键词、品类、品牌、作者或内容主题。
2. 汇总高相关笔记、评论反馈、受众关注点和高频表达方式。
3. 如果用户做运营，提炼选题方向、标题角度和内容结构。
4. 区分用户体验反馈、种草表达和真实可验证信息。

## 输出要求

- 默认按“选题概览 / 热门角度 / 评论反馈 / 可执行建议”输出。
- 如果是品牌调研，要额外标出负面反馈与风险点。
- 对不可访问内容直接说明。
