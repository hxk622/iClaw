# 东方财富金融数据 Skill

这个 Skill 用于把东方财富金融数据能力接入到 iClaw。

## 适用场景

- 查询股票、行业、板块、指数、基金、债券的实时行情
- 查询主力资金流向、估值、财务指标、高管与股东信息
- 查询企业经营、股权与高管关联关系
- 需要最新金融数据，而不是依赖模型过时知识时

## 安装前提

需要在东方财富妙想 Hub 获取 API Key，并在运行时通过环境变量 `EASTMONEY_APIKEY` 提供。

## 调用约定

- 查询接口：`POST https://mkapi2.dfcfs.com/finskillshub/api/claw/query`
- 请求头：
  - `Content-Type: application/json`
  - `apikey: ${EASTMONEY_APIKEY}`
- 请求体示例：

```json
{
  "toolQuery": "东方财富最新价"
}
```

## 输出要求

- 优先返回结构化 JSON 结果
- 解释关键字段含义，避免直接把原始字段名丢给用户
- 当数据范围过大时，提醒用户缩小时间范围或缩小标的范围
- 当结果为空时，提示用户到东方财富妙想 AI 核验

## 注意事项

- 谨慎查询大范围历史数据，避免上下文过大
- 输出时优先保留证券主体、指标名称、时间范围和数值单位
