# Market Sentiment Indicators

## 恐慌与贪婪指标

### VIX 恐慌指数
| VIX 水平 | 市场情绪 | 历史参考 |
|---------|---------|---------|
| <12 | 极度贪婪 | 市场过热警示 |
| 12-20 | 正常 | 常态区间 |
| 20-30 | 担忧 | 关注风险 |
| 30-40 | 恐慌 | 可能超跌 |
| >40 | 极度恐慌 | 历史底部区域 |

### CNN Fear & Greed Index
- 0-25: Extreme Fear (极度恐惧)
- 25-45: Fear (恐惧)
- 45-55: Neutral (中性)
- 55-75: Greed (贪婪)
- 75-100: Extreme Greed (极度贪婪)

## 美股情绪指标

### Put/Call Ratio
| P/C Ratio | 含义 |
|-----------|------|
| <0.7 | 过度乐观 (看涨过多) |
| 0.7-1.0 | 正常 |
| >1.0 | 过度悲观 (看跌过多) |
| >1.2 | 极度恐慌 |

### AAII Sentiment Survey
美国个人投资者协会情绪调查:
- Bullish > 50%: 过度乐观警示
- Bearish > 50%: 过度悲观 (逆向买入信号)
- Bull-Bear Spread > 30%: 极端乐观
- Bull-Bear Spread < -30%: 极端悲观

### Margin Debt
融资余额:
- 快速上升: 市场过热信号
- 高位回落: 可能见顶
- 低位企稳: 底部信号

## 情绪综合评分

将多个指标加权计算综合情绪分:

```
情绪分 = w1×VIX分 + w2×成交量分 + w3×涨跌比分 + w4×资金流向分

评级:
80-100: 极度贪婪 - 注意风险
60-80: 贪婪 - 乐观但需谨慎
40-60: 中性
20-40: 恐惧 - 开始关注机会
0-20: 极度恐惧 - 逆向思维机会
```
